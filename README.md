# Number Theory Computational Experiments

This repository contains Python scripts for generating and analyzing number-theoretic sequences. 
It includes programs to compute a combinatorial formula for numbers in a given range, 
focusing on odd primes (OEIS A065091) and many other OEIS sequences.
